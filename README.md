## Системные функции

| Название функции          | Описание                                                                 |
|---------------------------|-------------------------------------------------------------------------|
| `initSmoothScroll()`       | Плавная прокрутка к якорным ссылкам                                    |
| `handleMobileMenu()`       | Управление мобильным меню (toggle на 768px)                           |
| `animateOnScroll()`        | Анимация элементов при скролле с Intersection Observer                |
| `loadOptimizedImages()`    | Динамическая загрузка оптимизированных изображений                    |
| `handleHoverEffects()`     | Управление ховер-эффектами для навигации и изображений                |
| `initScrollIndicator()`    | Анимация индикатора прокрутки "Read more"                             |

## Инструкция по установке

1. Клонировать репозиторий:
```bash
git clone https://github.com/yourusername/mwaz-website.git
```

2. Установить зависимости:
```bash
npm install
```

3. Запустить dev-сервер:
```bash
npm run dev
```

4. Сборка для production:
```bash
npm run build
```

Требования к окружению:
- Node.js v16+
- npm v8+
- Modern browser with ES6+ support
